# SS14 Displacement Map Editor

A hopefully more intuitive way to edit SS14 displacement maps

## Description

This is a beginner webdev project and a prototype at that, so expect bugs, uninuitive behavior, and missing documentation. I'll be trying to clean this up and add documentation and make it more intuitive to use.

## Usage

This editor is a static webapp and is hosted at https://orbitalellie.github.io/ss14-displacement-editor/
